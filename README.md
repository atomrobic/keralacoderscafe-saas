# 🚀 Kerala Code Cafe SaaS

A community-driven open-source SaaS project built by developers from Kerala 🇮🇳

This project focuses on building real-world SaaS solutions like subscription management, automation systems, and business tools using modern technologies.

---

## 🌟 Vision

To build a strong developer community by creating scalable, real-world SaaS applications and learning together.

---

## 🛠️ Tech Stack

* 🔹 Backend: Django / Django REST Framework
* 🔹 Frontend: React.js
* 🔹 Database: PostgreSQL / SQLite
* 🔹 Automation: Python + Selenium
* 🔹 API: REST APIs

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/your-username/keralacodecafe-saas.git
cd keralacodecafe-saas
```

### 2. Backend Setup (Django)

```bash
cd backend
python -m venv venv
venv\Scripts\activate   # Windows
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

### 3. Frontend Setup (React)

```bash
cd frontend
npm install
npm run dev
```
